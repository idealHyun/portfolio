# Click Meee

---

### 📖 프로젝트 개요

Click Meee는 온라인상으로 클릭을 통해 다른 사용자와 대결할 수 있는 플랫폼입니다.
시각적인 차트와 대결을 통해 재미를 즐길 수 있습니다.

### ✨ 프로젝트 주요 기능

- 누구나 방 생성과 참여를 할 수 있습니다.
- 게임 중 클릭 수에 대한 실시간 차트 반영
- 게임 결과 후 보여지는 정리된 결과

### 🛠️ 사용 기술 스택

- **Frontend**: React, TypeScript, Tailwindcss, Recoil
- **Backend**: WebSocket, STOMP protocol

### ⚙️ 구현한 기능

- WebSocket 및 STOMP 프로토콜을 통해 클릭 데이터를 실시간 반영
- Chart.js를 통해 차트 구현
- 주고 받는 데이터를 통해 UI 구현
