# SOUL MBTIE

---

### 📖 프로젝트 개요

SOUL MBTIE는 **MBTI**와 **관심사**를 기반으로 사용자 간 매칭을 제공하는 소셜 네트워크 플랫폼입니다.
**공통점**을 강조하여 자연스럽게 대화를 시작할 수 있도록 돕고,
**실시간 채팅**과 **밸런스게임** 기능으로 소통의 재미를 더합니다.

### ✨ 프로젝트 주요 기능

- **사용자 매칭**: MBTI와 관심사가 유사한 사람을 매칭하여 공통점을 요약 제공하며, 간단한 메시지와 함께 친구 요청을 보낼 수 있음.
- **친구**: 친구 목록, 보낸 요청, 받은 요청 내역을 확인하고 관리할 수 있음.
- **채팅 및 밸런스게임**: 실시간 채팅과 밸런스게임을 통해 사용자 간 소통과 재미를 강화.

### 🛠️ 사용 기술 스택

- **Frontend**: Flutter,Dart, GetX, React, JavaScript
- **Backend**: Node.js, Socket.IO
- **Notification**: Firebase Cloud Messaging (FCM)

### ⚙️ 구현한 기능

- 실시간 채팅 및 밸런스게임 기능 구현 (Socket.IO 활용)
- 채팅 및 친구 요청 푸시 알림 구현 (FCM 활용)
- GetX를 이용한 상태 관리 및 UI 설계
- 친구 관리 페이지 및 관리자 페이지 개발

### 📚 학습과 성장

- **실시간 데이터 처리**: Socket.IO를 활용하며 심화 학습 및 문제 해결 경험.
- **예외 처리**: 소켓 연결 오류 해결을 통해 안정적인 서비스 설계 학습.
- **푸시 알림 구현**: FCM 기반 알림 시스템 설계 및 구현 능력 향상.
- **효율적 상태 관리**: GetX를 활용해 UI와 로직 관리 숙련.
