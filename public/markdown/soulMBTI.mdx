# SOUL MBTIE

---

### 📖 프로젝트 개요

SOUL MBTIE는 **MBTI**와 **관심사**를 기반으로 사용자 간 매칭을 제공하는 소셜 네트워크 플랫폼입니다.
**공통점**을 강조하여 자연스럽게 대화를 시작할 수 있도록 돕고,
**실시간 채팅**과 **밸런스게임** 기능으로 소통의 재미를 더합니다.

### ✨ 프로젝트 주요 기능

- 자신과 MBTI와 관심사가 비슷한 사람을 매칭
- 매칭인원과 어떤 점이 비슷한지에 간단 요약 설명 제공
- 채팅 중에 상대방과 밸런스게임 기능 추가

### 🛠️ 사용 기술 스택

- **Frontend**: Flutter,Dart, GetX, React, JavaScript
- **Backend**: Node.js, Socket.IO
- **Notification**: Firebase Cloud Messaging (FCM)

### ⚙️ 구현한 기능

- Socket.IO를 이용한 채팅 기능 및 밸런스게임 구현
- FCM을 통해 채팅 및 친구 요청 알림 구현
- GetX를 이용하여 상태 관리 구현
- 친구 관련 페이지 및 관리자 페이지 구현

### 📚 학습과 성장

- Socket.IO를 활용한 실시간 데이터 처리 기술 심화 학습 및 문제 해결 경험
- 소켓 연결 관련하여 오류를 잡으면서 예외 처리에 대한 중요성을 학습
- FCM 기반 푸시 알림 시스템 설계 및 구현
- GetX를 활용한 효율적인 상태 관리와 UI 설계 능력 숙련
