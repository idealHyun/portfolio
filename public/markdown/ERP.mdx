# ERP 프로그램

---

### 📖 **프로젝트 개요**
대학교 DB 관련 과목을 수강하며 처음으로 협업하여 진행한 프로젝트.

### ✨ **주요 기능**
- **로그인 및 사원 등록**: 사용자 인증 및 신규 사원 등록.
- **출근부 및 급여 명세서 관리**: 출근부 기록 및 추가 수당을 포함한 급여 명세서 작성.
- **결재 리스트 관리**: 결재 요청 및 진행 상태 관리.
- **일일 업무 관리**: 개인 및 팀별 업무 일정 관리.
- **쪽지 기능**: 사용자 간 쪽지 송수신 및 관리.

### 🛠️ **사용 기술 스택**
- **Frontend**: Windows Forms
- **Backend**: C#, MySQL

### ⚙️ **구현한 주요 기능**
- 사용자 간 쪽지 송수신 기능
- 쪽지 수신 시 실시간 알림 기능

### 📚 **학습과 성장**
- **SQL 이해도 향상**: 다양한 테이블을 조인하여 작성한 SQL 문을 통해 데이터베이스에 대한 이해도가 크게 향상됨.
- **Polling 개념 학습**: 실시간 알림 기능 구현 과정에서 Polling의 개념을 습득.
- **협업 도구의 중요성 인식**: Git을 사용하지 않고 코드 병합 방식으로 작업하며, Git과 같은 버전 관리 시스템의 필요성을 절감.
