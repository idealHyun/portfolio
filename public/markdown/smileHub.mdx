# Smile Hub

---

### 📖 프로젝트 개요

Smile Hub 는 멋쟁이 사자처럼 중앙 해커톤을 참여하면서 '건강' 이라는 주제로 만들게 된 웹 어플리케이션입니다.
사용자는 웃음을 기록하며 다양한 챌린지에 참여하고 업적을 달성할 수 있습니다.

### ✨ 프로젝트 주요 기능

- **웃음 인식 및 기록**: 카메라를 통해 웃음을 인식하고 웃음의 퍼센트를 기록
- **챌린지 및 업적**: 웃었던 것을 기록하여 챌린지나 업적을 달성하는 재미 요소 추가
- **친구**: 쉽게 친구를 초대하고 다른 사용자의 기록을 확인할 수 있어 선의의 경쟁 추가

### 🛠️ 사용 기술 스택

- **Frontend**: JavaScript, React, Sass, React-query, Antd
- **Backend**: Spring, Mysql
- **Devops**: AWS, Git Actions, Netlify
- **Etc**: Notion, Figma

### ⚙️ 구현한 기능

- 실시간 채팅 및 밸런스게임 기능 구현 (Socket.IO 활용)
- 채팅 및 친구 요청 푸시 알림 구현 (FCM 활용)
- GetX를 이용한 상태 관리 및 UI 설계
- 친구 관리 페이지 및 관리자 페이지 개발

### 📚 학습과 성장

- **실시간 데이터 처리**: Socket.IO를 활용하며 심화 학습 및 문제 해결 경험.
- **예외 처리**: 소켓 연결 오류 해결을 통해 안정적인 서비스 설계 학습.
- **푸시 알림 구현**: FCM 기반 알림 시스템 설계 및 구현 능력 향상.
- **효율적 상태 관리**: GetX를 활용해 UI와 로직 관리 숙련.
